
# API GO

Bootcamp de programación en GO, ejercicio de aplicación de creación de un API.

## Tech Stack

**Server:** Go

**Database:** Postgres

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DB_HOST`

`DB_PORT`

`DB_NAME`

`DB_USER`

`DB_PASSWORD`

## Run Locally

Start the server

```bash
  go run main.go
```
